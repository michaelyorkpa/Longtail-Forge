Reusable lists are normal list records marked as reusable. Keep starter grocery lists, packing lists, project supply lists, parts starters, and on-site checklists there, then create an independent active working copy when it is time to execute the workflow.

Duplicating a list copies the useful structure and resets working progress such as checked state, completion state, purchase status, tracking IDs, and actual costs. Item suggestions come from explicit catalog-backed or explicitly saved reusable items. Catalog changes affect future suggestions and do not rewrite historical list items.
