Lists preserve the context needed to continue work: item progress, next unchecked item, earliest needed-by date, last list or item activity, source list lineage, and permission-safe linked records.

This context helps users recover after interruption without turning Lists into Dashboard, Workbench, Tasks, Notes, Files, Search, or published reference content. Completed, finalized, archived, and deleted states preserve useful history while active incomplete lists stay focused on the next useful action.
