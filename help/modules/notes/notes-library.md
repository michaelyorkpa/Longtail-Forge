The Notes Library groups notes by current purpose. Active Work is for notes tied to work in progress. Ongoing Areas is for durable areas that stay active over time. Reference Library is for retained material. Archive keeps completed notes available for historical lookup while separating them from active Library counts.

Library counts, collection counts, and search filters only expose notes the current user can read. Moving a note between active buckets may clear its collection when the old collection belongs to a different bucket.
