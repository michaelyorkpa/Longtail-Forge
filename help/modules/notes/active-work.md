Active Work is the bucket for notes tied to current action, open tasks, active tickets, and short-term working context.

Task or ticket context can suggest Active Work when a note is created, but the suggestion is only a starting point. The note still follows its visibility, owner, linked-record access, collection, secure-note, and permission rules.

Active Work notes can provide supporting context for resumable work when the current user can read them. Private and secure notes do not expose body previews in supporting-context payloads.
