# Sample Data

This folder contains generic sample data for trying the Time Tracker app without using real client information.

To use it, copy:

- `sample-data/data/client-project.json` to `data/client-project.json`
- `sample-data/data/settings.json` to `data/settings.json`
- `sample-data/data/time-entries.csv` to `data/time-entries.csv`
- `sample-data/logs/app-events.csv` to `logs/app-events.csv`

Restart the local server after replacing files.
